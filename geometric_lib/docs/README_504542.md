# Общее описание решения

Каждый файл содержит функции, получающие на вход данные о фигуре, 
возвращающие площадь, периметр нужной фигуры

+ **circle.py** - операции с окружностью/кругом
+ **square.py** - операции с квадратом

# Исходный код модулей

## circle.py

```python
import math

def area(r):
    '''
    Принимает радиус окружности r
    Возвращает площадь окружности πr**2
    '''
    return math.pi * r * r

def perimeter(r):
    '''
    Принимает радиус окружности r
    Возвращает периметр окружности 2πr
    '''
    return 2 * math.pi * r
```

## square.py

```python
def area(a):
    '''
    Принимает сторону квадрата a
    Возвращает площадь квадрата a**2
    '''
    return a * a

def perimeter(a):
    '''
    Принимает сторону квадрата a
    Возвращает периметр квадрата 4a
    '''
    return 4 * a
```

# Описание функций

## Файл circle.py

### Функция area
Функция принимает на вход радиус окружности (r) <br/>
и возвращает её площадь (πr²)

**Пример:**
```python
area(3)  # вернёт 9π ≈ 28.27433388
```

### Функция perimeter
Функция принимает на вход радиус окружности (r) <br/>
и возвращает её периметр/длину (2πr)

**Пример:**
```python
perimeter(3)  # вернёт 6π ≈ 18.84955592
```

## Файл square.py

### Функция area
Функция принимает на вход сторону квадрата (a) <br/>
и возвращает его площадь (a²)

**Пример:**
```python
area(3)  # вернёт 9
```

### Функция perimeter
Функция принимает на вход сторону квадрата (a) <br/>
и возвращает его периметр (4a)

**Пример:**
```python
perimeter(3)  # вернёт 12
```
